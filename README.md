# TASK App - Kotlin Android Application

The **TASK App** is a task management application designed to help users organize, track, and complete their daily tasks efficiently. Developed in **Kotlin** using **Android Studio**, this app leverages modern Android development principles, including Jetpack libraries and MVVM architecture.

---

## Features

- **Task Management**: Add, update, delete, and mark tasks as completed.
- **Categorization**: Organize tasks into customizable categories.
- **Reminders**: Set reminders to stay on top of your tasks.
- **Search and Filter**: Quickly find tasks using the search and filter functionalities.
- **Dark Mode**: Supports a dark theme for better user experience.
- **Offline Access**: Manage tasks even without an internet connection.

---

## Technologies Used

### Languages and Frameworks
- **Kotlin**: Primary programming language.
- **Android SDK**: For Android development.

### Architecture
- **MVVM (Model-View-ViewModel)**: Ensures separation of concerns and scalability.

### Libraries and Tools
- **Jetpack Components**: Room, LiveData, ViewModel, and Navigation.
- **Coroutines**: For asynchronous operations.
- **Material Design**: Provides a modern and intuitive user interface.
- **Hilt/Dagger**: Dependency injection for modular and testable code.
